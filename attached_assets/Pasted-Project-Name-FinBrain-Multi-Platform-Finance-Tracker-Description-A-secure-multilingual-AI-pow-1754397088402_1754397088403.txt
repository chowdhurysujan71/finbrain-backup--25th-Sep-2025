Project Name: FinBrain - Multi-Platform Finance Tracker

Description:
A secure, multilingual, AI-powered expense logging app that tracks user spending via WhatsApp and Facebook Messenger. The system automatically extracts amounts from messages, categorizes expenses, stores them in PostgreSQL, and calculates monthly summaries in real-time.

Core Features
Flask Web Server with /health and /webhook endpoints

PostgreSQL Database with three tables:

expenses (detailed expense log)

users (hashed user profiles)

monthly_summaries (analytics and insights)

Secure User Identification

SHA-256 hashing for all user IDs (phone numbers or PSIDs)

No raw phone numbers stored in the database

Intelligent Expense Logging

Extracts amounts with regex

Categorizes into 10 categories (Food, Transport, Shopping, etc.)

Updates monthly totals automatically

Unified Webhook

Handles WhatsApp (Twilio) and Facebook Messenger messages

Responds to users with a logged confirmation message

Environment Variables for secure credentials

DATABASE_URL

TWILIO_ACCOUNT_SID / TWILIO_AUTH_TOKEN / TWILIO_WHATSAPP_NUMBER

FACEBOOK_PAGE_ACCESS_TOKEN / FACEBOOK_VERIFY_TOKEN

SESSION_SECRET

Technical Requirements
Python 3.11+

Flask for web server

psycopg2-binary for PostgreSQL

Twilio & Requests for WhatsApp integration

Hashlib for SHA-256 user anonymization

dotenv for environment variable management
